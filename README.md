Word Shooting
====

## Requirement
console-size 80x24  
java  
bash  

## Install and Usage
$ git clone https://github.com/koshitake2m2/word_shooting  
$ cd word_shooting/src  
$ chmod +x run read_key.sh  
$ ./run
